# Food Image Classification using MobileNetV2

## Overview
This project builds a multi-class image classification model to identify 101 food categories using deep learning and transfer learning.

## Tech Stack
- Python
- TensorFlow / Keras
- NumPy, Pandas
- Matplotlib, Seaborn
- Scikit-learn

## Model
- Pretrained MobileNetV2
- Fine-tuned on Food-101 dataset

## Results
- Test Accuracy: ~43%
- Evaluated using confusion matrix and classification report
